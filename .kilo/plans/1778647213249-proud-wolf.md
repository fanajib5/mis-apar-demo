# Rencana: Produktisasi APAR Management System — 3 Mode Deployment

## Ringkasan Bisnis

Proyek ini akan diubah dari proyek kustom klien (PT. Penyedia APAR) menjadi **produk demo** yang bisa dipresentasikan ke calon pembeli dan dijual kembali (resell). Tiga opsi deployment memberikan fleksibilitas harga sesuai kemampuan pasar.

---

## 1. Salin Dokumentasi

Salin `docs/mis-apar-init.md` ke `docs/mis-apar-demo.md`, lalu perluas dengan:

- Visi produk (bukan proyek klien)
- 3 mode deployment dengan penjelasan, arsitektur, harga, dan target pasar masing-masing
- Strategi penjualan & reseller
- Demo script / use case demo
- Fitur khusus APAR: inspeksi berkala, tracking kadaluarsa/belum servis

## 3. Tiga Mode Deployment

### Mode A: Full Online (SaaS)

| Aspek | Detail |
|-------|--------|
| **Arsitektur** | Laravel Cloud / VPS + Inertia React SPA |
| **Database** | PostgreSQL (cloud) |
| **Offline** | Tidak ada — butuh internet |
| **Multi-tenant** | Ya, pakai sistem Team yang sudah ada |
| **Infrastruktur** | Rp 200-500k/bulan |
| **Harga Jual** | Rp 200-500k/bulan (SaaS) |
| **Target Pasar** | Kota besar dengan internet stabil, budget kecil |
| **Cocok untuk** | Perusahaan APAR yang sudah punya kantor & internet bagus |

### Mode B: Hybrid (Online + Offline with Sync)

| Aspek | Detail |
|-------|--------|
| **Arsitektur** | Cloud server (Laravel) + PWA / Electron / Tauri desktop app |
| **Database** | PostgreSQL (cloud) + SQLite (local) via sync |
| **Offline** | Bisa: input data offline, sinkronisasi saat online |
| **Sync Engine** | Bidirectional push/pull dengan conflict resolution (last-write-wins / server priority) |
| **Multi-tenant** | Ya |
| **Infrastruktur** | Rp 500-750k/bulan (server) + Rp 2-5jt (aplikasi desktop) |
| **Harga Jual** | Rp 5-7jt (sekali) + Rp 200k/bulan maintenance |
| **Target Pasar** | Perusahaan dengan internet tidak stabil, butuh akses 24/7 |
| **Cocok untuk** | Kantor cabang, tim lapangan, daerah dengan internet terbatas |

### Mode C: Full Offline / Local First

| Aspek | Detail |
|-------|--------|
| **Arsitektur** | Desktop app mandiri (Electron/Tauri + PHP embedded via Laragon/XAMPP bundle) |
| **Database** | SQLite lokal sepenuhnya |
| **Online** | Opsional — hanya untuk backup/restore, tidak untuk operasional |
| **Sync** | Manual export/import file (pilihan) atau backup cloud opsional |
| **Multi-tenant** | Tidak — 1 instalasi = 1 perusahaan |
| **Infrastruktur** | Rp 0 (server) — hanya laptop/PC klien |
| **Harga Jual** | Rp 3-5jt (sekali lisensi) |
| **Target Pasar** | Perusahaan tanpa internet, budget kecil, atau keamanan data tinggi |
| **Cocok untuk** | Daerah terpencil, perusahaan yang tidak mau data di cloud |

## 4. Strategi Penjualan & Reseller

| Aspek | Strategi |
|-------|----------|
| **Demo** | Mode A (full online) sebagai demo — langsung jalan tanpa setup. Jika klien suka, upgrade ke B atau C sesuai kebutuhan. |
| **Reseller** | Beri akses demo (Mode A) + diskon reseller 20-30% untuk dijual kembali. Sediakan white-label (ganti logo, nama perusahaan). |
| **Trial** | 14 hari gratis Mode A (full online), konversi ke berbayar. |
| **Upgrade Path** | A → B: tambah fitur offline. A/B → C: beli lisensi standalone. |
| **Pricing Tiers** | A: Rp 200-500k/bulan · B: Rp 5-7jt + Rp 200k/bulan · C: Rp 3-5jt sekali |

8. **Pricing & billing** — subscription management
